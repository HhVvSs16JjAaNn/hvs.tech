Engineering Intern

KONIGTRONICS (OPC) Pvt. Ltd. | Aug 2024 - Oct 2024

Internship Overview

This internship provided hands-on experience in the development and testing of cutting-edge IoT and smart home technologies. I was deeply involved in the product development lifecycle, from hardware design and firmware integration to building user-facing control systems. The primary focus was on creating practical, efficient solutions for smart switching and load sensing applications.

Key Responsibilities & Contributions

  • Smart Switching & Load Sensing: Contributed to the core development of smart switching devices. This involved working with microcontrollers and sensors to accurately detect and manage electrical loads, a foundational component for energy management and automation systems.
  • Autonomous Lighting System: Led the design and validation of an autonomous room lighting system. This proof-of-concept project utilized IR sensors to detect human presence and wireless modules for communication, creating a responsive and energy-efficient lighting solution. I was responsible for both the hardware setup and the logic controlling its behavior.
  • Firmware & Software Integration: Played a key role in integrating embedded firmware with a Raspberry Pi. I developed and deployed control logic that allowed the Raspberry Pi to act as a central hub, processing sensor data and managing the smart devices.
  • Web-Based Control Dashboards: Built user-friendly control dashboards using the Flask web framework. These dashboards provided a simple interface for monitoring sensor status, controlling devices remotely, and visualizing data, demonstrating the feasibility of a complete end-to-end IoT system.
  • Technical Documentation: Created comprehensive, client-facing documentation outlining the system architecture, setup procedures, and final project deliverables. This ensured clarity and facilitated knowledge transfer to stakeholders.

Technologies Used

Python Flask Raspberry Pi Embedded C IoT IR Sensors Load Sensing Tech Wireless Modules

Learning Outcomes

This role significantly strengthened my skills in rapid prototyping, full-stack IoT development, and hardware-software co-design. I gained practical experience in translating conceptual requirements into functional hardware and software, managing project timelines, and producing professional-grade documentation for technical and non-technical audiences.
