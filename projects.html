Featured Project

User-Friendly Access & Optimization of Ground Operations for UAS

Engineered a system to automate on-ground drone operations, integrating a Raspberry Pi with a Pixhawk flight controller to reduce manual dependencies. Developed Python scripts using the MAVLink protocol to automate mission uploads, pre-flight checks, and arming sequences via a physical button interface. Successfully reduced mission initiation time by 50%.
